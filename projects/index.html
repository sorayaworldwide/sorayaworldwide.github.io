Machine Learning

Sentiment Classification of Twitter Sentiment: Covid-19, Vaccines, and Education

This machine learning project sought to classify the sentiment of tweets based on textual characteristics of the data. The analysis was conducted in R using the TidyModels framework using multinomial regression. The project was heavily influenced by Emil Hvitfelt and Julia Silge's excellent resource Supervised Machine Learning for Text Analysis in R.
